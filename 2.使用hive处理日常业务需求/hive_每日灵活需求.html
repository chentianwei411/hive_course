题目一：每个用户截止到每月为止的最大交易金额和累计到该月的总交易金额，结果数据格式如下

思路：

  • 1.对数据按照客户及其年-月分组
  • 2.分组后就每月销售金额之和
  • 3.使用窗口函数，对每个客户不同月份分组求最大值(max)和累计值(sum)

 

题目二：计算用户的回购率和复购率

复购率: 当前月份购买2次及以上的客户占所有客户比例

回购率：当前月份购买且上个月份也购买的客户占当月所有月份客户比例

思路：

复购率

  • 1、对当月（2月份）的客户分组，计数购买次数
  • 2、筛选购买次数为2以上的，认为是复购群体

回购率

  • 1、筛选当月及上月部分
  • 2、利用客户id进行当月连上月，推荐左连
  • 3、对同一条客户id均有购买记录的，认为是回购群体

题目一：计算每个用户截止到每月为止的最大交易金额和累计到该月的总交易金额

第一步：提取需要字段及按客户id、年月分组，求分组后的订单量及消费金额

select customer_key,substr(create_date,1,7) as umonth,count(sales_order_key) as ucount,sum(unit_price) as usum from ods_sales_orders group by customer_key,substr(create_date,1,7) limit 10;

第二步：利用窗口函数，对客户按照月份排序，求最大金额及累积金额。

xxxxxxxxxx
select customer_key,umonth,ucount,
max(usum) over(partition by customer_key order by umonth) as current_max,
sum(usum) over(partition by customer_key order by umonth) as current_sum
from
(select customer_key,substr(create_date,1,7) as umonth,count(sales_order_key) as ucount,sum(unit_price) as usum from ods_sales_orders group by customer_key,substr(create_date,1,7)) as t1 limit 10;

参考: Hive分析窗口函数(一) SUM,AVG,MIN,MAX https://www.cnblogs.com/qingyunzong/p/8782794.html

HIVE over() 超全讲解 https://blog.csdn.net/qq_22222499/article/details/92406370

 

题目二：计算用户的回购率和复购率

复购率:

xxxxxxxxxx
select count(ct),count(if(ct>1,1,null)),count(if(ct>1,1,null))/count(ct) as ratio from(
select customer_key,count(customer_key) as ct from  ods_sales_orders
where month(create_date) = 2
group by customer_key) t;

回购率:

第一步：

xxxxxxxxxx
select * from 
(
(select  customer_key,substr(create_date,1,7) as umonth from ods_sales_orders
group by customer_key,substr(create_date,1,7)) a
left join (select  customer_key,substr(create_date,1,7) as umonth from ods_sales_orders
group by customer_key,substr(create_date,1,7)) b on a.customer_key = b.customer_key and substring(a.umonth,6,2) =  (substring(b.umonth,6,2) -1) ) limit 10;

第二步：

xxxxxxxxxx
select a.umonth,count(a.customer_key) as mcount,count(b.customer_key) as lcount,concat(round((count(b.customer_key)/count(a.customer_key))*100,2),"%") as ratio from 
(
(select  customer_key,substr(create_date,1,7) as umonth from ods_sales_orders
group by customer_key,substr(create_date,1,7)) a
left join (select  customer_key,substr(create_date,1,7) as umonth from ods_sales_orders
group by customer_key,substr(create_date,1,7)) b on a.customer_key = b.customer_key and substring(a.umonth,6,2) =  (substring(b.umonth,6,2) -1) ) group by a.umonth;

 

题目三：求用户号对应不同的产品

第一步：

xxxxxxxxxx
SELECT  customer_key ,
                    cpzl_zw,
                    row_number() over(partition by customer_key order by create_date asc) as order_num
from ods_sales_orders;

第二步：

xxxxxxxxxx
with tmp as(
select 
            customer_key,
            cpzl_zw,order_num,cpzl_zw1 
        from  
            (SELECT  customer_key ,
                    cpzl_zw,
                    row_number() over(partition by customer_key order by create_date asc) as order_num,
                    lag(cpzl_zw,1,null) OVER(partition by customer_key order by create_date asc) AS cpzl_zw1 from ods_sales_orders) as a 
    where cpzl_zw != cpzl_zw1),
tmp2 as (
select customer_key,cpzl_zw,order_num,cpzl_zw1,
row_number() over(partition by customer_key order by order_num ) as cpzl_zw_num
from tmp
)select concat( customer_key,'-',concat_ws('-', collect_set(cpzl_zw)) )
from tmp2 where cpzl_zw_num <3
group by customer_key;

 

题目四：查询每年5月份购买过的顾客及总人数

#over 指定函数工作的数据窗口大小

xxxxxxxxxx
select customer_key,
count(*) over() 
from ods_sales_orders
where month(create_date)="5"
group by customer_key;

题目六：上述的场景,要将unit_price 按照日期进行累加

xxxxxxxxxx
select *,sum(unit_price) over(sort by create_date rows between UNBOUNDED PRECEDING and CURRENT ROW)

xxxxxxxxxx
select *, 
lag(create_date,1) over(distribute by customer_key sort by create_date) from ods_sales_orders limit 10;

 

题目八：查询最近前20%时间的订单信息

第一步：用ntile函数将订单时间按顺序分为5堆

xxxxxxxxxx
select *, ntile(5) over(sort by create_date desc) gid from ods_sales_orders limit 10;

第二步：取第一堆

xxxxxxxxxx
select * from (
    select cplb_zw,cpzl_zw,unit_price,create_date, ntile(5) over(order by create_date desc) sorted
    from ods_sales_orders
) t
where sorted = 1;
